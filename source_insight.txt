My tunning:
1. Change almost all fonts
   Options / Document Options / Screen Options

2. There are still some fonts not good, tune for each
   Mark the ones you want to modify
   Right click on them and choose "Style Properties"
   choose type name and change fonts
   Example:
   "Auto Annotation" is the one that Source Insight gives recommentation

3. Show line number
   Options / Document Options
   Choose "Show line numbers"
